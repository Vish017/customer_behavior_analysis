# customer_behavior_analysis
End-to-end analysis of 3,900 retail transactions using Python, PostgreSQL, and Power BI to uncover customer spending patterns, segmentation, and product insight.

📊 Data Analytics Project
📌 Overview

This project demonstrates an end-to-end data analytics workflow — from data loading and cleaning to analysis and visualization. The goal is to extract meaningful insights from raw data and present them through an interactive Power BI dashboard and a structured report.

📂 Dataset
The dataset contains structured data related to business operations (e.g., sales, customers, or transactions).
It includes multiple features such as categorical and numerical variables.
Data was sourced from a CSV/Excel file and loaded into Python for processing.
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI (Dashboard creation & visualization)
Gamma (Presentation creation)
🔄 Project Workflow
1. Data Loading
Imported dataset using Python (Pandas)
Initial inspection using .head(), .info(), .describe()
2. Exploratory Data Analysis (EDA)
Identified patterns, trends, and anomalies
Used visualizations like:
Bar charts
Histograms
Correlation heatmaps
3. Data Cleaning
Handled missing values
Removed duplicates
Standardized column formats
Fixed inconsistent data entries
4. SQL Analysis
Loaded cleaned data into database
Performed SQL queries such as:
Aggregations (SUM, COUNT, AVG)
Joins
Window functions
Ranking and filtering
5. Dashboard Creation (Power BI)
Built interactive dashboard with:
KPIs
Filters & slicers
Charts (bar, line, pie, etc.)
Focused on user-friendly and business-oriented insights
6. Reporting & Presentation
Created a detailed report summarizing findings
Designed a professional presentation using Gamma
📊 Dashboard

The Power BI dashboard includes:

Key performance indicators (KPIs)
Trend analysis
Category-wise comparisons
Interactive filters for better user experience
📈 Results & Insights
Identified key trends and patterns in the data
Highlighted top-performing categories/segments
Provided actionable insights for decision-making
Improved data quality through cleaning and preprocessing
